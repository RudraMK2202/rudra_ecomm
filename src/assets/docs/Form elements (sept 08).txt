                               Form Elements
                    ======================================

        - TextBox
        - Password
        - Number

                                  Range Input
                               -------------------

- It similar to a number input but will display a slider to select values.
- All attributes are same as Number input.

Syntax:
<input type="range"  min="1" max="100" value="50" step="5">


                                Email Input
                          ----------------------
- It validates email address.

Syntax:
    <input type="email" .... >

                                URL Input
                         ------------------------
- It validates standard URL format.
- URL format must have
        protocol://address

Syntax:
     <input type="url">

                                 Color Input
                      ------------------------------
- It can display a color panel.
- You can pick and select a color.

Syntax:
    <input type="color">

- You can set any default color value by using 6 chars hexa code
           
        #RRGGBB

Syntax:
    <input type="color" value="#ff0000">

Ex:
<form>
      <div>
         <figure>
            <img src="../public/Images/jacket.jpg" width="100" height="100">
            <figcaption>
               <input type="color" value="#f00">
            </figcaption>
         </figure>
      </div>
   </form>


                               Date and Time Input
                        -------------------------------

- It allows to pick a date and time.
- HTML provides various types for date and time
        1. date
        2. time
        3. datetime-local
        4. month
        5. year
        6. week
- You can't select range of dates.
- You can restrict date range.

Syntax:
    <input type="date">            allows to select any date.

     <input type="date" min="yyyy-mm-dd"  max="yyyy-mm-dd">
    <input type="date" min="2022-09-08" max="2022-09-24">

                                  File Input

- It is used to browse and select a file from client device.

Syntax:
    <input type="file">

- You can set filters for browsing but can't restrict.

       <input type="file"  accept=".jpg, *.pdf">

- By default you can select only one file, if you want multiple files to select then define the attribute  "multiple".

Syntax:
      <input type="file" multiple  accept=".jpg">


                                 Radio Buttons
                      --------------------------------------

- Radio button allows to select one or multiple options from a group of choices.
- Radio button once checked can't be unchecked.
- Radios are used with "Mutex" mechanism. [Mutual Exclusion]
- To enable Mutext you have to define a common name for radios in a group.

Syntax:
        <input type="radio" name="gender"> Male
        <input type="radio" name="gender"> Female

- You can make any radio selected by default by using "checked" attribute.

        <input type="radio" name="gender" checked> Male
       
- Every radio by default submit "on" as value. Hence every radio must have a reference value.

Ex:
 <form>
      <fieldset>
         <legend>Choose Gender</legend>
         <input name="gender" value="Male" checked type="radio"> Male
         <input name="gender" value="Female" type="radio"> Female
         <br><br>
         <button>Submit</button>
      </fieldset>
   </form>
