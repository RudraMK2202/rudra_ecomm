Bootstrap Classes for Form Elements

1. Install Bootstrap in your project

        >npm install  bootstrap  --save

2. Bootstrap CSS file is available in

      "node_modules/bootstrap/dist/css/bootstrap.css"

    Note:  "dist" is folder that contains files ready for "Production" [Go Live]

3. Link Bootstrap icons and Bootstrap CSS into webpage

    <link rel="stylesheet" href="../node_modules/bootstrap/dist/css/bootstrap.css">

    <link rel="stylesheet" href="../node_modules/bootstrap-icons/font/bootstrap-icons.css">

4. Bootstrap class used for Form Elements

Class Name                        Description
----------------------------------------------------------------------------------------------

                                    textarea.

form-select                        dropdown, listbox

form-check-input                    radio and checkbox

input-group-lg|sm                large size.

Ex: Designing Form with standard Format


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <link rel="stylesheet" href="../../node_modules/bootstrap/dist/css/bootstrap.css">
    <link rel="stylesheet" href="../../node_modules/bootstrap-icons/font/bootstrap-icons.css">
    <style>
        body {
            display: flex;
            justify-content: center;
            align-items: center;
            height: 500px;
        }
        form {
            border:1px solid gray;
            padding: 30px;
            width: 400px;
        }
    </style>
</head>
<body>
   <form>
      <h3> <span class="bi bi-person-fill"></span> Register User</h3>
      <div class="mb-3">                                                  [every category-->keep inside a div(container)]
          <label class="form-label" for="UserName">User Name</label>      ["label" defines the heading of the category
                                                                            it works as a '<span>']
          <div>                                                           [to make the "label" work for a particular input, use
                                                                           an attribute "for= 'name' of the input" inside label]
            <input type="text" class="form-control" name="UserName">
          </div>
      </div>
      <div class="mb-3">
         <label class="form-label" for="Password">Password</label>
         <div>
            <input type="password" class="form-control" name="Password">
         </div>
      </div>
      <div class="mb-3">
         <label class="form-label" for="City">Your City</label>
         <div>
            <select name="City" class="form-select">
                <option>Select Your City</option>
                <option>Delhi</option>
                <option>Hyd</option>
            </select>
         </div>
      </div>
      <div class="mb-3">
         <label class="form-label" for="Subscribe">Subscribe</label>
         <div  class="form-switch">
            <input type="checkbox" checked class="form-check-input" name="Subscribe"> <label>Yes</label>
         </div>
      </div>
      <div>
        <button class="btn btn-primary w-100">Register</button>
      </div>
   </form>
</body>
</html>
